Características del Sistema de Roles
Funcionalidades Implementadas:
  • ✅ Navegación dinámica por rol
  • ✅ Control de acceso a páginas
  • ✅ Elementos UI basados en permisos
  • ✅ Indicadores de rol visual
  • ✅ Logout automático
  • ✅ Redirección por rol al login
  • ✅ Página de acceso denegado
  • ✅ Persistencia de sesión
Jerarquía de Roles:
  1. Administrador - Acceso completo
  2. HR Manager - Gestión de RRHH
  3. Reclutador - Gestión de candidatos
  4. Hiring Manager - Gestión de empleos
  5. Empleado - Acceso limitado
  6. Candidato - Solo portal público
